## PHOTOSHOP WORKFLOW

### Pozadí varianta A — produktové fotky (odstřih)
1. Select Subject → Refine Edge (Smooth 3, Feather 1px)
2. Nová vrstva pod: Fill `#F1EEE5`
3. Radiální gradient: střed `#FAFBF8`, okraj `#E7E1D6`
4. Drop Shadow: Distance 4, Spread 0, Size 12, Opacity 8%

### Pozadí varianta B — lifestyle/workshop (color grade)
1. Camera Raw: Temperature +8, Tint +3
2. Highlights -15, Shadows +20
3. Clarity +10, Dehaze +5
4. HSL: Orange saturation -10, Luminance +5
5. Color Grading: Shadows #8B7355, Highlights #FFF8E7

### Export
- Formát: **WebP**, kvalita **85%** (parallax 80%)
- Vždy exportuj **@1x** i **@2x** (dvojnásobek rozměru)
- Pojmenování: `nazev-souboru.webp` a `nazev-souboru@2x.webp`
- Cílová složka: `/public/images/`
